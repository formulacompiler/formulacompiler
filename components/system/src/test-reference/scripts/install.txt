To install a script for OpenOffice (refTestSheets.py)

1. Run installOOoScript.sh on Unix or installOOoScript.bat on Windows.


To inxtall a script for MS Excel (refTestSheets.bas)

1. Create personal.xls in the "%APPDATA%\Microsoft\Excel\XLSTART" folder. On Windows XP %APPDATA% is usually "C:\Documents and Settings\%USERNAME%\Application Data". You can also put macros in any other .xls file.
2. Open Visual Basic Editor (Tools -> Macro -> Visual Basic Editor).
3. Import refTestSheets.bas (File -> Import File)
4. Save the .xls file.
