Contains the public API of SEJ, with the most important types being {@link sej.SpreadsheetLoader}, {@link sej.Compiler}, {@link sej.Engine}, and {@link sej.Engine.Computation}. For most uses, you should be able to confine yourself to the types defined here. The one notable exception is the registration of the compiler and spreadsheet loader implementations to be used.
