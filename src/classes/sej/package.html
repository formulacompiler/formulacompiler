Contains the public compile-time API of SEJ. All of the functionality is accessed using the factory methods in {@link sej.SEJ}. This class inherits from {@link sej.runtime.SEJRuntime}, which contains only the elements necessary for loading and using engines at run-time.

For most uses, you should be able to confine yourself to the types defined here and in {@link sej.runtime}.
