The Spreadsheet Engine for Java (SEJ, for short) makes it possible to use computations defined in spreadsheet files within normal Java applications (see the Design Document for a quick example). The original spreadsheet files can be created, for example, in Microsoft Excel or OpenOffice Calc. SEJ read the spreadsheet files using the {@link sej.SpreadsheetLoader} and builds corresponding Java-based formula evaluation engines using a {@link sej.Compiler}. Such an {@link sej.Engine} computes output values given a set of input values defined by a {@link sej.Computation}. For example, compute the value of the cell B3 given the values of the cells B1 and B2 as inputs.
