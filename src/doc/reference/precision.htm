Numeric Precision In SEJ

If you set a fixed number of decimal places after the point on a cell in Excel, SEJ reflects this setting when computing final outputs. Like Excel, it does not limit the precision on intermediate results.

Here's the handling of cell precision, shown by example (the number format is shown in small type behind the cell value or formula):

Precision
AB
212.0=12.34 (#0.)
312.3=12.34 (#0.0)
412.34=12.34 (#0.00)
512.34=12.34 (#0.000)

Rounding
AB
712.0=12.0049 (#0.00)
812.01=12.005 (#0.00)
913.0=12.999 (#0.00)
1012.01=12.0149 (#0.00)
1112.02=12.015 (#0.00)
1212.0=12.0001 (#0.00)
13-12.0=-12.0049 (#0.00)
14-12.01=-12.005 (#0.00)
15-13.0=-12.999 (#0.00)
16-12.0=-12.0001 (#0.00)

Intermediate values have full precision
