SEJ Tutorial

A first draft of a tutorial for the Spreadsheet Engine for Java (SEJ). The tutorial is organized as a collection of use cases, which I mean to extend as new questions and examples crop up.

Binding Data
How to bind the input and output cells of a spreadsheet to methods of plain Java interfaces or classes.
Numeric Types
How to make an engine use, for instance, BigDecimal instead of Double for all numeric computations - great for financial applications.
